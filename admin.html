🔑 API Configuration
Set your Anthropic API key here. It will be used for all email generation requests.
Anthropic API Key
Get your API key from console.anthropic.com → API Keys → Create Key.
Format: sk-ant-api03-… (starts with sk-ant-)
How to get an Anthropic API Key
Follow these steps:
1Go to console.anthropic.com
2Create an account or sign in
3Click API Keys in the left menu
4Click Create Key → copy the key
5Paste it in the field above and click Save

📖 Docs & Help
How to connect your frontend and deploy this backend.
Connect Your Frontend
In your index.html, change these two lines at the top of the script:
// Change these two values: const USE_BACKEND = true; const BACKEND_URL = 'https://your-railway-url.railway.app';
Deploy on Railway (Free)
Easiest way to host this backend online:
1Push this backend folder to GitHub
2Go to railway.app → New Project → Deploy from GitHub
3Add env variable: ANTHROPIC_API_KEY=sk-ant-…
4Add env variable: ADMIN_PASSWORD=your-secret
5Railway auto-deploys — you get a public URL
